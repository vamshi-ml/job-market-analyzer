
# Job Market Analyzer

Analysis of data science job postings to identify trends, salary patterns, and in-demand skills.

## What it does:
- Analyzes 2,500+ job postings
- Identifies most common job titles
- Shows salary ranges by experience level
- Reveals top hiring countries
- Visualizes key insights

## Tools used:
- Python
- Pandas
- Matplotlib
- Kaggle dataset

## Key findings:
- Average salary: $133,409
- Most jobs for: Senior Engineers
- Top hiring country: United States
- Highest paid experience: Executive level
